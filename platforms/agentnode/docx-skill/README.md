# DOCX Skill - Document Creation and Manipulation

## Overview

The DOCX Skill is AgentNode's highest-monetization-potential agent, providing comprehensive document creation, editing, and analysis capabilities for professional-grade Microsoft Word document processing. This skill enables AI agents to programmatically create, modify, and analyze .docx files with full support for complex formatting, tracked changes, comments, and document structure manipulation.

## Key Features

### Document Creation
- **Full Document Generation**: Create complete .docx documents from scratch with professional formatting
- **Template-Based Creation**: Use pre-defined templates for consistent document generation
- **Multi-Section Support**: Create documents with multiple sections, headers, and footers
- **Table Integration**: Insert and format complex tables with merged cells, borders, and styling

### Document Editing
- **Content Modification**: Insert, replace, and delete text while preserving formatting
- **Paragraph Manipulation**: Adjust spacing, alignment, indentation, and paragraph styles
- **Style Management**: Apply and modify document styles for consistent formatting
- **Image Handling**: Insert, resize, and position images within documents

### Advanced Features
- **Tracked Changes**: Enable, accept, and reject tracked changes with full revision history
- **Comments System**: Add, edit, and manage document comments and annotations
- **Document Merging**: Combine multiple documents into a single cohesive file
- **Format Conversion**: Convert documents to various formats (PDF, HTML, plain text)

### Document Analysis
- **Text Extraction**: Extract clean text content while ignoring formatting
- **Structure Analysis**: Identify document sections, headings, and hierarchical structure
- **Metadata Extraction**: Retrieve document properties, author information, and revision history
- **Content Search**: Find and highlight specific text patterns within documents

## Technical Specifications

### Supported Formats
- Input: .docx, .doc, .dotx, .dotm
- Output: .docx, .pdf, .html, .txt, .odt

### Document Limits
- Maximum file size: 100 MB
- Maximum pages: 10,000
- Maximum paragraphs: 100,000
- Maximum embedded images: 500 per document

### API Performance
- Document creation: 2-5 seconds average
- Complex editing: 1-3 seconds average
- Text extraction: <1 second average
- Batch processing: Up to 100 documents per request
