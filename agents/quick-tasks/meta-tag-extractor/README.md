# Meta Tag Extractor

> **Quick Task Agent - SEO Analysis**

## Overview
Extract and analyze all meta tags from any webpage. Perfect for SEO audits and competitive analysis.

## Capabilities
- Title extraction
- Description extraction
- Open Graph tags
- Twitter Card tags
- Viewport settings
- Canonical detection

## Input
```json
{
  "url": "https://example.com",
  "include_og": true,
  "include_twitter": true
}
```

## Output
```json
{
  "title": "Example Domain",
  "description": "Example domain for...",
  "og:title": "Example",
  "og:image": "https://...",
  "twitter:card": "summary",
  "canonical": "https://example.com/",
  "issues": ["Missing og:description"]
}
```
