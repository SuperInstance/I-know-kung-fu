# Password Generator

> **Quick Task Agent - Security Utility**

## Overview
Generate secure, memorable passwords with customizable complexity requirements.

## Capabilities
- Random passwords
- Memorable passphrases
- PIN generation
- Complexity rules
- Password strength meter
- Breach checking

## Input
```json
{
  "type": "random|passphrase|pin",
  "length": 16,
  "include_symbols": true,
  "include_numbers": true,
  "check_breach": true
}
```

## Output
```json
{
  "password": "K9#mP2$vL5nQ8xW",
  "strength": "very strong",
  "breached": false,
  "entropy": 95
}
```
