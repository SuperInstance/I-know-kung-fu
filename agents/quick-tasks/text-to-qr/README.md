# Text to QR Code Agent

> **Quick Task Agent - QR Generation**

## Overview
Generate QR codes from any text, URL, or data. Customizable colors, logos, and formats.

## Capabilities
- URL QR codes
- vCard QR codes
- WiFi QR codes
- Custom colors
- Logo embedding
- Multiple formats (PNG, SVG)

## Input
```json
{
  "text": "https://example.com",
  "type": "url|vcard|wifi|text",
  "color": "#000000",
  "bg_color": "#FFFFFF",
  "logo": "optional logo URL"
}
```

## Output
```json
{
  "qr_image": "base64 or URL",
  "format": "png",
  "size": "300x300",
  "scan_test": "passed"
}
```
