# EXIF Data Viewer

> **Quick Task Agent - Image Metadata**

## Overview
View, edit, and strip EXIF metadata from images. Privacy-focused for photographers and social media.

## Capabilities
- Full EXIF extraction
- GPS data viewing
- Camera settings display
- Metadata stripping
- Date/time editing
- Privacy cleanup

## Input
```json
{
  "image": "file or URL",
  "action": "view|strip|edit",
  "remove_gps": true,
  "remove_all": false
}
```

## Output
```json
{
  "camera": "Canon EOS R5",
  "settings": {"iso": 400, "f": 2.8},
  "gps": {"lat": 37.7749, "lng": -122.4194},
  "taken": "2024-03-15 14:32:00",
  "has_metadata": true
}
```
