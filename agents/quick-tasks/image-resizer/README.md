# Image Resizer Agent

> **Quick Task Agent - Image Processing**

## Overview
Resize images to any dimension while maintaining aspect ratio and quality. Perfect for web, social, and apps.

## Capabilities
- Custom dimensions
- Aspect ratio preservation
- Batch processing
- Format conversion
- Quality optimization
- Watermarking

## Input
```json
{
  "image": "file or URL",
  "width": 800,
  "height": 600,
  "maintain_aspect": true,
  "format": "webp"
}
```

## Output
```json
{
  "original_size": [1920, 1080],
  "new_size": [800, 450],
  "format": "webp",
  "file_size_kb": 45
}
```
