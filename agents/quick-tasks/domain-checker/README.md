# Domain Availability Checker

> **Quick Task Agent - Domain Research**

## Overview
Check domain availability across multiple TLDs instantly. Essential for brand naming and startup planning.

## Capabilities
- Single domain check
- Bulk domain check
- Multiple TLD suggestions
- Premium domain identification
- Historical availability

## Input
```json
{
  "domain": "example.com",
  "check_tlds": [".com", ".io", ".co", ".ai"],
  "bulk": ["domain1.com", "domain2.com"]
}
```

## Output
```json
{
  "domain": "example.com",
  "available": false,
  "price": "$12.99",
  "alternatives": ["example.io", "example.co"],
  "premium": false
}
```
