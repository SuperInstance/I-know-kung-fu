# WHOIS Lookup Agent

> **Quick Task Agent - Domain Intelligence**

## Overview
Query WHOIS data for any domain. Get registration details, nameservers, and contact information.

## Capabilities
- Full WHOIS data
- Registration dates
- Nameserver info
- Registrar identification
- Privacy detection
- Historical records

## Input
```json
{
  "domain": "example.com",
  "include_history": false
}
```

## Output
```json
{
  "domain": "example.com",
  "registrar": "GoDaddy",
  "created": "1995-08-13",
  "expires": "2025-08-12",
  "nameservers": ["ns1.example.com"],
  "status": "clientTransferProhibited",
  "privacy_protected": true
}
```
