# IP Geolocator Agent

> **Quick Task Agent - Location Intelligence**

## Overview
Geolocate IP addresses to get location, ISP, and network information. Perfect for security and analytics.

## Capabilities
- Country/region/city
- ISP identification
- ASN information
- VPN/Proxy detection
- Bulk IP lookup
- Risk assessment

## Input
```json
{
  "ip": "8.8.8.8",
  "bulk": ["1.1.1.1", "8.8.8.8"]
}
```

## Output
```json
{
  "ip": "8.8.8.8",
  "country": "United States",
  "city": "Mountain View",
  "isp": "Google LLC",
  "asn": "AS15169",
  "is_vpn": false,
  "risk_score": 0
}
```
