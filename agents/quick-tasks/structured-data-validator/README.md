# Structured Data Validator

> **Quick Task Agent - SEO Schema**

## Overview
Validate JSON-LD structured data against Google's schema requirements. Essential for SEO.

## Capabilities
- Schema.org validation
- Google Rich Results check
- Multiple schema types
- Error highlighting
- Fix suggestions
- Preview snippets

## Input
```json
{
  "url": "https://example.com",
  "schema_type": "Article|Product|LocalBusiness"
}
```

## Output
```json
{
  "valid": true,
  "schema_type": "Article",
  "errors": [],
  "warnings": ["author field recommended"],
  "rich_results_eligible": true
}
```
