## 🔄 Standard Workflow

```
1. RECEIVE TASK
   ↓
2. ASSESS COMPLEXITY
   ├── Simple → TOOL_CARDS.md → Quick Tasks section
   ├── Moderate → DECISION_TREE.md → Reasoning branch
   ├── Complex → DECISION_TREE.md → Multi-Agent branch
   └── Enterprise → NAVIGATION.md → Combined Suites
   ↓
3. LOCATE AGENT
   ├── Check README.md for capabilities
   ├── Review configuration options
   └── Note example usage
   ↓
4. CONFIGURE & EXECUTE
   ├── Apply configuration from agent spec
   ├── Execute with provided patterns
   └── Monitor for issues
   ↓
5. ITERATE IF NEEDED
   ├── Check RELATIONSHIPS.md for complementary agents
   ├── Combine if beneficial
   └── Document results
```
